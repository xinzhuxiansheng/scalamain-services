## Functional Programing in Scala (Scala函数式编程)  

**引用透明**    
引用透明要求函数不论进行了任何操作都可以用它的返回值来代替。 理解引用透明的另一种方式是引用透明的表达式不依赖上下文，可以本地推到，而那些非引用透明的表达式是依赖上下文的，而且需要全局推导。 


**异常**
1.异常破坏了引用透明并引入了上下文依赖
2.异常不是类型安全的
